# Index

A powerful unified search engine that aggregates and indexes your data across multiple platforms
using vector embeddings and natural language processing.

## Overview

Index connects to various APIs (Discord, Google Drive, Gmail, Notion) to collect and index your
documents. It converts these documents into vector embeddings and stores them in Pinecone, enabling
you to search across all your connected data sources using natural language queries.

## Features

- **Multi-platform Integration**: Connect Discord, Google Drive, Gmail, and Notion
- **Vector Embeddings**: Documents are converted to vector embeddings for semantic search
- **Natural Language Search**: Search your data using plain English queries
- **Unified Interface**: One search interface for all your connected data sources

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Desktop**: Tauri
- **Backend**: Rust
- **Vector Database**: Pinecone
